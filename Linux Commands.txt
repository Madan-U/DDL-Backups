1) LS COMMAND
   ----------
-> To list all files and folders.
-> To list files with detailed information.

# To list all files and folders in the current working directory : 
  ls

# To list all files and folders present in a specific directory : 
  ls Documents/

# To list all files and folders present in home directory : 
  ls ~

# To list all files and folders of parent directory of the present working directory : 
  ls ../

# Go to two step back : 
  ls ../.. 

# To list all files in long format : 
  ls -l

# To list all files and folders including hidden files :
  ls -a

# To list all files and folders including hidden files in long format :
  ls -al  

# To sort the files based on their size 
  ls -lS

# To move the results of ls command to another file (.out.txt)
  ls -lS > out.txt

# To list only directory
  ls -d /*

16) CHMOD COMMAND
    -------------

-> The chmod command in Linux is used to change the permissions of a file.
-> -rwx (User) -rw (Group users) -r (Other Users). 
-> r = Read, w = write, x = execute.


# To grant read write execute permission to a other users :
  chmod o+rwx filename
  
# To grant only read permission to other users :
  chmod o+r filename 

# To revoke all permission from other users :
  chmod o-rwx filename 

# To grant and revoke permissions from group :
  chmod g+rwx filename 
  chmod g-rwx filename 

# To grant and revoke permissions from user :
  chmod u+rwx filename
  chmod u-rwx filename

17) DF, DU, FREE COMMANDS 
    ---------------------

-> The df, du, and free commands in Linux are used to display information about disk and memory usage.

# DF COMMAND : Reports the amount of disk space used and available on file systems.
  ----------
  df -h 
	-h  : To get human readble form of output.
	
# DU COMMAND : Estimates file space usage. It provides a summary of disk usage for files and directories.
  ----------
  du -sh 	
	-sh : Summarizes disk usage of the current directory in short form.

# FREE COMMAND : Displays the amount of free and used memory in the system. 
  ------------
  free    : Displays memory usage.
  free -h : Displays memory usage in a human-readable format.
  free -m : Displays memory usage in megabytes.
  free -g : Displays memory usage in gigabytes.

25) GREP COMMAND
    ------------
# The grep command in Linux is used to search for text patterns within files.

# syntax : grep [options] pattern [file...]

# Example : grep "pattern" filename.ext

# FIELDS 
  "UNIX" is a pattern.

# -c :  To count number of lines that matches the given pattern.
	grep -c "UNIX" file.txt

# -h :  To display lines that matches the given pattern.
	grep -h "UNIX" file.txt

# -l :  To display all the files that contains the given pattern.
	grep -l "UNIX" 

# -n :  To display all the matching lines with line numbers 
	grep -n "LINUX" file.txt

# -v :  To display all non matching lines 
	grep -v "UNIX" file.txt

# -o  : To display all the matching patterns 
	grep -o "UNIX" file.txt

# -e  : To search for multiple patterns 
	grep -e "LINUX" -e "UNIX" file.txt

# -^  : To find the lines which are starts with the given pattern
	grep ^UNIX file.ext

# -$  : To find the lines which are ends with the given pattern
	grep $UNIX file.txt

# -i  : To find matching lines with case insestive
	grep -i "UNIX" file.ext
